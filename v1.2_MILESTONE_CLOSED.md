# Milestone 1.2 - Release Version
Fecha: 2026-04-19

## Estado: Cerrado - Estable

### Resumen de Cambios Clave:
1. **Refinamiento de UI:** 
   - Colores de prioridad en Kanban optimizados para visibilidad.
   - Etiquetas de estado en Tracker acortadas y protegidas contra wrap.
   - Nuevo Favicon SVG vectorial implementado.
2. **Internacionalización (i18n):**
   - Traducciones completas en Español (es_AR, es_ES), Inglés (en_US, en_GB) y Portugués (pt_BR, pt_PT).
3. **Automatización de Despliegue:**
   - Script `prepare.ts` para empaquetado automático.
   - Configuración de `.htaccess` para ruteo en subcarpetas de cPanel.
   - Gestión automática de credenciales en `api/config.php`.
4. **Documentación:**
   - Diccionario de datos, arquitectura y manuales generados en carpeta `/documentacion`.

### Nota Técnica:
Este estado corresponde al **Checkpoint actual de la plataforma**. Para asegurar la persistencia externa, se recomienda utilizar la función "Export to ZIP" o "Sync to GitHub" desde el menú de la aplicación.
